# 廣義相對論的核心是潮汐力？ ― 從自由落體理解時空的彎曲！

把兩顆球並排同時放手，它們會平行落下、永遠不會碰到一起？
其實嚴格來說並非如此。在這個小小疑問的前方，正等待著廣義相對論的核心概念——「時空的彎曲」。
