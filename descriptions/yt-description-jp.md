# 一般相対論の核心は潮汐力？ ― 時空の曲がりを自由落下から理解しよう！

2つのボールを横に並べて同時に落としたら、平行に落ちてぶつからない？
実はこれ、厳密にはちょっと違います。この小さな疑問の先に、一般相対論の核心「時空の曲がり」が待っています。

この動画では、自由落下でも消すことのできない「潮汐力」を手がかりに、時空の曲がり(リーマン曲率)とは何かを数式なしで直感的かつ視覚的に解説します。

🔍 この動画でわかること：
・ 自由落下で重力が「消える」仕組み ― 等価原理
・ それでも消せない重力の正体 ― 潮汐力
・ 潮汐力と時空の曲がり(リーマン曲率)のつながり
・ 曲がった空間での平行移動が起こす不思議な現象

🔔 この動画はこんな人におすすめ
・ 一般相対論に興味があるけど数式が苦手な人
・ 「時空の曲がり」のイメージがつかめない人
・ 潮汐力や重力の仕組みをもっと知りたい人
