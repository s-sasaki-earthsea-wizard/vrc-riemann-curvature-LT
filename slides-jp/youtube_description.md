# 一般相対論は潮の満ち引きからわかる？　〜時空の曲がりの直感的理解〜

ブラックホールに近づくと体がスパゲッティのように引き伸ばされる「スパゲッティ化」。
実はこれと同じ現象が、今この瞬間、地球上のあなたの体にも起きています。

この動画では潮の満ち引きを起こす「潮汐力」という身近な現象から、アインシュタインの一般相対論の核心である「時空の曲がり(リーマン曲率)」を直感的に理解するための解説をします。

🔍 この動画でわかること：
・ 自由落下で「消せる重力」と「消せない重力」の違い
・ 潮汐力とは何か、なぜ海の潮の満ち引きが起きるのか
・ 潮汐力は時空の曲がりで引き起こされる？
・ 曲がった時空では「まっすぐ進んでも交わる」理由
